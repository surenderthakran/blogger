MY PROJECTS

  • GoMind

    GoMind is a simplistic Neural Network library written entirely in Go (GoLang).
    It serves its purpose as a simplistic library for quick training in Go without worrying about C bindings or pre-creating models in Python.

  • Dockerized Go Environemnt

    A dockerized development and production environment for Go to help jumpstart a new project.
    A combination of Docker and Makefile creates a live reloading development environment which performs test runs and auto formatting on every reload.

  • Dockerfiles

    Managed set of dockerfiles for Node.js, MongoDB, Go, Nginx, MySQL etc on my custom base images.
